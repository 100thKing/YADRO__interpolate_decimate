**Задание**

1. Написать на Matlab или Python генератор синуса на частоте f = 0..500 Гц при фиксированной частоте дискретизациив 1 кГц
2. Написать на выбранном языке функцию для интерполяции(повышения частоты дискретизации) входного воздействия в 3 раза. Можно использовать любой известный для выполняющего задание метод
3. написать на выбранном языке функцию для децимации(понижения частоты дискретизации) входного воздействия в 3 раза. Можно использовать любой известный для выполняющего задание метод
4. Пропустить входное воздействие с генератора синуса последовательно через дециматор и интерполятор. Измерить ошибку между входными и выходнымми данными в зависимости от f. Теоретически обьяснить полученный результат

**decimate.m**

```
% Децимация (понижение частоты дискретизации) в M раз с предварительной антиалиасинговой фильтрацией
%
% Метод: FIR-фильтрация + прореживание (downsampling)
%
%   Этапы:
%     1) Антиалиасинговый низкочастотный FIR-фильтр с частотой среза Wc = 1/M
%        (нормированная частота, 0..1, где 1 = Fs/2).
%        Фильтр удаляет частоты выше новой частоты Найквиста Fs/(2*M),
%        чтобы избежать aliasing при прореживании.
%     2) Downsampling: берём каждый M-й отсчёт: y = x_filtered[1::M]
%
%
% Входные параметры:
%   x - входной вектор отсчётов
%   M - коэффициент децимации (по умолчанию 3)
%
% Выходные параметры:
%   y - децимированный вектор, длина length(x)/M
```

**interpolate.m**

```
% Интерполяция (повышение частоты дискретизации) в M раз методом линейной интерполяции
%
% Метод: линейная интерполяция
%   Между каждой парой соседних отсчётов вставляется (M-1) новых отсчётов,
%   значения которых вычисляются линейно:
%       y[k*L + m] = x[k] + m/M * (x[k+1] - x[k]),  m = 0,1,...,M-1
%
%   Это эквивалентно:
%     1) вставке M-1 нулей между отсчётами (upsampling),
%     2) фильтрации FIR-фильтром с линейной импульсной характеристикой.
%
% Входные параметры:
%   x - входной вектор отсчётов
%   M - коэффициент интерполяции (по умолчанию 3)
%
% Выходные параметры:
%   y - интерполированный вектор, длина M * length(x)
```

**Обьяснение полученного результата**

По теореме Котельникова(она же теорема Найквиста-Шеннона) - частота дискретизации сигнала должна быть **не менее** чем удвоенная максимальная частота сигнала

В данной задаче частота дискретизации - 1 кГц = 1000 Гц

По теореме Котельникова - максимальная частота сигнала должна не превышать 1000\2 = 500 гЦ, иначе возникнет алиасинг

Для данной задачи использовались дециматор и интерполятор с коэффициентом = 3

При такой децимации новая частота дискретизации становиться равна 1000\3 = 333.3 гЦ, а значит и частота сигнала не должна превышать 333.3\2 = 167 гЦ (по теореме Котельникова).

При низких значениях частоты сигнал сохраняется достаточно точно, поскольку он лежит в полосе пропускания используемых фильтров и не нарушает условия дискретизации.
